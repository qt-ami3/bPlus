##  Loops & functions; braces:

One bracket on func init line and then return like; *Can be ignored for 2 line for loop*

```
#include <iostream>

void useless_crap() {
  for (int i = 0; i < 10; i++)
    std::cout << i << endl;
}

int main() {
  int x = 0;

  while (true) {
    x++;
    useless_crap;
    if (x == 1) {
      break;
    }
  }

  cout << std::endl;

  return 0;
}
```
